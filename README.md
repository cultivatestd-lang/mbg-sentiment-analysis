# Analisis Sentimen MBG

Aplikasi Streamlit untuk klasifikasi sentimen tweet berbahasa Indonesia terkait Program Makan Bergizi Gratis (MBG) menggunakan TF-IDF + Support Vector Machine (kernel linear).

## Menjalankan secara lokal

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy ke Streamlit Community Cloud

1. Buka https://share.streamlit.io
2. New app -> pilih repository ini, branch `main`, file `app.py`
3. Klik Deploy
